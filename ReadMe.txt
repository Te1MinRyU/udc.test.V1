This is to manipulate geojson data to show the local geoinformation of Utsunomiya.

Including:

1. 4 data layers
2. 4 icons
3. A JavaScript file
4. A html file
5. A css file

Note: 

Due to the Security policy of some brokers such as Google Chrome, the geojson data may not be read into the web page. It is still under construction of modifying the relative data path in order to improve the functionalities.
